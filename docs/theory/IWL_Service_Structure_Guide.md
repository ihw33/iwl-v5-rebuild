# 🎯 IWL v5.0 서비스 구조 이해 가이드

> 이 문서는 IWL v5.0 서비스의 전체 구조와 흐름을 이해하기 위한 가이드입니다.
> 새로운 AI 어시스턴트나 팀원이 프로젝트에 참여할 때 반드시 읽어야 할 문서입니다.

---

## 📊 전체 서비스 흐름

```
설문조사 → 사고성향 진단 → 모듈 조합 추천 → 35분 수업 진행 → 피드백 & 다음 추천
```

### 단계별 설명
1. **설문조사**: 학습자의 니즈와 현재 수준 파악 (2-5단계 적응형)
2. **진단**: 4축 사고 성향 분석 (A1~A4)
3. **모듈 조합**: 8×4 매트릭스에서 최적 모듈 3개 선택
4. **수업 진행**: B0 템플릿 기반 35분 실습
5. **피드백**: 학습 데이터 수집 및 다음 경로 추천

---

## 🔧 핵심 구성 요소 (4가지)

### 1. 설문 시스템
**목적**: 평가가 아닌 니즈 파악
**특징**:
- 2-5단계 적응형 설문
- 분기 로직으로 개인화
- Junior는 보호자 설문 포함

**관련 문서**:
- `/docs/templates/Survey_System_Complete.md` - 전체 시스템
- `/docs/templates/B0_Survey_System_Revised.md` - B0용 설문

### 2. 8×4 매트릭스 (32개 모듈)
**구성**: 8단계(Stage) × 4축(Axis) = 32개 조합
```
S1~S8: 지각인지 → 철학적 인지 (사고의 깊이)
A1~A4: 정보처리, 사고조작, 추상화, 자기인식 (인지 기준)
```

**관련 문서**:
- `/docs/theory/8x4_Matrix_Theory_Guide.md` - 이론 설명
- `/docs/GLOSSARY.md` - 용어 정의

### 3. 모듈 조합 원칙
**방식**: 페르소나 + 목표 → AI가 3개 모듈 자동 조합
**예시 조합**:
- 초보자: S1-A1 + S2-A1 + S3-A2 (기초 순차)
- 중급자: S3-A2 + S5-A3 + S4-A1 (균형 발전)
- 고급자: S6-A4 + S7-A3 + S8-A4 (메타인지)

**관련 문서**:
- `/ai_prompts/execution_prompts.md` - 자동 조합 프롬프트
- `/ai_prompts/system_prompts/5_stage_prompt_chain.md` - AI 체인

### 4. B0 표준 템플릿
**구조**: 이중 구조 (Frontend/Backend)
- **Frontend**: 학습자용 - 이론 숨김, 실습만
- **Backend**: 제공자용 - 측정, 분석, 관리

**관련 문서**:
- `/docs/templates/B0_Standard_Template.md` - 메인 템플릿
- `/docs/templates/B0_Example_Tech_Proposal.md` - 실제 예시
- `/docs/templates/B0_Template_Schema.json` - 데이터 구조

## 💡 실제 예시: 김대리 케이스

### 1. 설문 입력
```yaml
직업: 마케팅팀 대리 (3년차)
경험: ChatGPT 가끔 사용 (번역, 자료조사)
니즈: 보고서 작성 시간 단축
어려움: 뭘 물어봐야 할지 막막, 답이 너무 일반적
```

### 2. 진단 결과
- **수준**: AI 초급 사용자 (L2)
- **성향**: A1(논리) 높음, A3(협업) 낮음
- **갭**: 니즈(보고서) vs 경험(번역만)

### 3. 모듈 추천
```
옵션 1 (기초 강화):
- S2-A1: 구체적 질문법
- S3-A1: 맥락 제공법
- S4-A2: 창의적 활용

옵션 2 (실무 직결):
- S3-A1: 보고서 구조화
- S5-A1: 데이터 분석
- S4-A4: 인사이트 도출
```

### 4. 수업 설계 (35분)
- **도입 (5분)**: "10분 만에 주간 보고서 초안 만들기"
- **실습 1 (10분)**: 보고서 구조 AI에게 제안받기
- **실습 2 (10분)**: 데이터를 인사이트로 변환하기
- **실습 3 (8분)**: 상사 관점으로 검토하기
- **마무리 (2분)**: 템플릿 저장, 다음 주 활용법

### 5. 결과
- 실제 보고서 초안 1개 완성
- 재사용 가능한 프롬프트 3개 획득
- 다음 추천: S6-A2 (창의적 제안 추가)

## 🎮 주요 원칙

### 절대 원칙
1. **35분 제약**: 모든 수업은 35분 내 완료
2. **실습 중심**: 이론 설명 최소화
3. **AI 도구 필수**: ChatGPT/Claude 등 활용
4. **산출물 지향**: 매 수업 실제 결과물 생성

### 설계 원칙
1. **모듈 숨김**: 학습자는 S1-A1 같은 용어 모름
2. **자연스러운 체득**: 활동을 통해 인지 능력 향상
3. **개인화**: 페르소나별 맞춤 설계
4. **데이터 기반**: 모든 활동 측정 및 분석
