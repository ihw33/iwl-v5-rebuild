## 📌 핵심 변경사항

### 1️⃣ **작업 확인**
- `DAILY_TASKS.md` 파일 하나로 관리
- GitHub Issue 댓글 확인 불필요

### 2️⃣ **작업 진행**  
- Thomas가 각 팀과 직접 작업
- 실시간으로 진행

### 3️⃣ **완료 처리**
- 완료 = 자동 승인 (별도 승인 절차 없음)
- DAILY_TASKS.md에 체크(✅)만 하면 끝

### 4️⃣ **기록**
- 하루 끝에 GitHub Issue 일괄 업데이트
- PM Claude가 처리

## 📝 팀별 작업 확인 방법

### **모든 팀 공통**
1. `DAILY_TASKS.md` 파일 열기
2. 자기 팀 섹션 확인
3. [ ] 표시된 작업 진행
4. 완료시 Thomas가 ✅ 체크

### **예시**
```markdown
## Cursor-기획팀
✅ Issue #21: A1/A3 개선 (완료)
[ ] Issue #13: B0 템플릿 작성  ← 이거 하면 됨
```
